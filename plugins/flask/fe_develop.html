# 插件结构说明

添加的插件结构如下:

.
├── package.json     # 插件基础配置
├── stage-config.js  # 舞台配置
├── README.md        # 给插件使用者查看的帮助文档
├── views            # 舞台页面
│   ├── xxx.vue
│   └── ...
├── components       # 组件
│   ├── xxx.vue
│   └── ...
└── assets           # 资源文件
    └── ...

查看文件结构说明插件结构

# 创建插件

  1. 进入项目文件夹
  2. 执行命令 npm run plugin-new 并根据提示操作
  3. 若项目未启动, 执行 npm run serve 启动项目

打开项目, 可以看到左侧栏中多了刚创建的插件内容, 并且可以点击打开对应页面. 接下来就可以根据具体业务需求进行开发了
